Coding Cheatsheet

HTML

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

Tag Tag usage

<html> HTML Element

All our HTML code is to be included under <html> element

 
<html>
  ...
  ...
</html>

<body> Body Element

Anything that needs to be displayed in browser is included under <body> element

 
<body>
  ...
  ...
</body>

<h1><h2>...<h6> Header Element

Text to be identified as header in browser is included under <h1><h2>...<h6> element

 
<h1>Largest font header</h1>
<h2>Larger font header</h2>
<h3>Large font header</h3>
<h4>Small font header</h4>
<h5>Smaller font header</h5>
<61>Smallest font header</h6>

<p> Paragraph Element

Text to be identified as paragraph in browser is included under <p> element

 
<p>This is a paragraph.</p>

<li> List Item Element

This element is used to list items inside:

  • Ordered Lists <ol>
  • Un-ordered Lists <ul>

 
<ol>
  <li>First item in list</li>
  <li>Second item in list</li>
</ol>

<ul>
  <li>Bulleted first item in list</li>
  <li> Bulleted second item in list</li>
</ul>
